# FormTransitionHistory Component

## Description

The FormTransitionHistory component is a Vue component used to display
transition history information in a list format.

## Props

| Name | Type | Default | Description |
| ---- | ---- | ------- | ----------- |
| `rendererProps` | ControlElement  | - | The properties passed down from the parent renderer. |

## Slots

This component does not have any slots.

## Directives

This component does not use any directives.

## Events

This component does not emit any events.

## Usage

The component is not used in any templates directly.
It's used by the `FormTransitionHistoryRenderer` to be rendered based on the control element from JSONForms.
